Etiquetas importantes

  • Tabla (para crear un tabla)
  • Tr para crear un fila Tabla Row
  • th para crear la fila de cabecera Tabla head por defecto aparacen en negrita y centradas
  • td para crear una celda de fila Table Data
  • caption Para poner titulo a una tabla

    • Etiqutas añadidas en Html5

  • thead Agrupacion de cabecera
  • tbody Cuerpo de la tabla
  • tfood Agrupacion del pie
